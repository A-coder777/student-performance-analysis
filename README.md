# Student Performance Analysis System

## 📌 Objective
This project analyzes student data to understand how study hours and attendance affect academic performance. It involves data cleaning, feature engineering, and data analysis.

## 📌 Dataset Description
- StudyHours: Number of hours studied per day
- Attendance: Attendance percentage
- Marks: Student marks

## 📌 Steps Performed
- Data Loading using pandas
- Data Cleaning (handling missing values and removing outliers)
- Feature Engineering (Performance & EffortScore)
- Data Analysis and Group Analysis

## 📌 Key Insights
1. Higher study hours generally improve marks.
2. Students with high attendance perform better.
3. Low attendance leads to poor performance.
4. EffortScore helps identify consistent students.
5. Study hours alone are not enough without attendance.
